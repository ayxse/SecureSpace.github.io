Secure Space

Cloud Server & Backup Services

Secure Space es un Cloud Server rentable, seguro y con un solo clic tienes función de sus necesidades de almacenamiento de copia de seguridad. La interfaz de administración centralizada facilita la definición de políticas de respaldo y protege una amplia gama de cargas de trabajo empresariales, incluye Maquinas Virtuales, bases de datos SQL y SAP entre otros recursos compartidos.

Consistencia de la aplicación

Realice copias de seguridad y restaure datos de máquinas virtuales con consistencia de aplicaciones en Windows usando Volume Shadow Copy Service (VSS) y en Linux con scripts de procesamiento previo y posterior.

Mas Información

Opciones de almacenamiento duraderas

Almacene sus copias de seguridad en almacenamiento con redundancia local (LRS), almacenamiento con redundancia geográfica (GRS) y almacenamiento con redundancia de zona (ZRS).
